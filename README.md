# 👋 ¡Bienvenidos! a mi proyecto bloc de contraseñas
<img src="imagen_presentacion.png" alt="Presentación" width="220" align="left" style="margin-right: 20px; border-radius: 5px;">
<p style="text-align: justify;">
    ¡Bienvenido a este repositorio! Aquí encontrarás la presentación de mi proyecto <b>Bloc de Contraseñas</b>, una propuesta enfocada en la gestión, organización y administración de credenciales digitales mediante diferentes tecnologías y enfoques de programación. Este proyecto fue desarrollado como ejercicio de aprendizaje utilizando <b>PSeInt, Python, Java, Visual Basic, Frontend y Backend</b>, permitiendo fortalecer conocimientos relacionados con lógica de programación, estructuras de datos, desarrollo web y manejo de bases de datos.
</p>
<p style="text-align: justify;">
    A lo largo del proyecto se implementaron diferentes versiones del mismo sistema, comenzando desde aplicaciones de consola hasta una aplicación web completa con almacenamiento local mediante <b>LocalStorage</b> y persistencia de datos utilizando <b>PHP, MySQL y phpMyAdmin</b>. De esta manera, el repositorio se convierte en una práctica integral orientada al fortalecimiento de habilidades en programación, desarrollo frontend, backend e integración de tecnologías.
</p>
<br clear="left"/>

### 📂 Tecnologías utilizadas en este proyecto
* PSeInt
* Python
* Java
* Visual Basic
* Frontend
* Backend

#
### ✨ Características del proyecto
* Gestión organizada de contraseñas y credenciales digitales.
* Desarrollo progresivo del mismo ejercicio en múltiples lenguajes de programación.
* Aplicaciones de consola e interfaz gráfica web.
* Almacenamiento local usando <b>LocalStorage</b>.
* Integración completa entre <b>Frontend</b> y <b>Backend</b>.
* Implementación de bases de datos utilizando <b>xampp y phpMyAdmin</b>.
* Operaciones CRUD funcionales.
* Actualización dinámica de registros en tiempo real.
* Sistema de categorías para organizar contraseñas:
    * Redes sociales
    * Correo electrónico
    * Bancos
    * Trabajo
    * Juegos
* Panel de indicadores dinámicos.
* Manipulación dinámica del DOM usando JavaScript.
* Comunicación entre JavaScript y PHP mediante <b>Fetch API</b>.
* Manejo de datos utilizando formato <b>JSON</b>.
* Proyecto desarrollado sin frameworks.
* Organización estructurada de archivos y carpetas.
* Enfoque orientado al aprendizaje y fortalecimiento de habilidades tecnológicas.
